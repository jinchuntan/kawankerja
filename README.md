# KawanKerja AI

KawanKerja AI is a demo-ready Next.js prototype for underserved ASEAN youth to discover scholarships, courses, internships, and jobs through personalized recommendations and a mock AI guidance assistant.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Local browser storage for profile and admin-edited listings
- Mock AI guidance with seeded demo opportunities

## Features

- Landing page with problem statement, features, impact metrics, and outreach plan
- Onboarding flow that saves a learner profile locally
- Dashboard with recommendation sections for scholarships, courses, internships, and jobs
- AI assistant page with preloaded prompts and simple mock responses
- Opportunity detail page with simplified summary and match explanation
- Admin page to add or edit demo opportunity listings locally

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open `http://localhost:3000`

## Notes

- All data is prototype/demo data.
- No real authentication or backend is included.
- User profiles and admin edits are stored in local browser storage only.
